INÍCIO CALCULADORA ESTATÍSTICA

EXPLICAÇÃO

Média Aritmética Simples: É a soma de todos os valores dividida pela quantidade de valores, útil para calcular um valor médio simples.

Média Aritmética Ponderada: Similar à média simples, mas cada valor pode ter um peso diferente, influenciando mais ou menos no resultado final.

Mediana: É o valor do meio em uma lista de números ordenados, ou a média dos dois valores centrais se houver um número par de elementos.

Moda: Representa o valor mais frequente em um conjunto de dados, útil para identificar a ocorrência mais comum dentro de uma distribuição.
